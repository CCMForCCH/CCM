Create a New Problem Report


Best Practices:
  • First, check to see that the problem hasn’t already been reported
  • Don’t combine multiple problems into one report unless absolutely necessary
  • Make the name of the problem unique, even if you have to use a date, e.g. "No hot water, 12/25/2023"
  • Give enough details to make the problem clear to others
  • Focus on symptoms, not solutions

Note about problem priorities:
  • Text messages are sent to all residents for critical-priority problems. (Use sparingly!)
  • Text messages are sent to the emergency team for emergency-priority problems.
  • Text messages are sent to the Service Coordinators and Service Manager for high-priority problems.
